# .agentoppa/candidates/ — 정정 큐 (비워지는 백로그)

`self-harden`이 포착한 정정 후보가 잠깐 머무는 곳. 분류·라우팅이 끝나면 **삭제(드레인)** — 영구 기록은 라우팅 산출물(validator 룰·픽스처·reference·AGENTS.md 줄)과 git이 가진다.

- 여긴 *지식베이스가 아니다* — 비어 있는 게 정상(미처리 백로그만 잠깐 쌓임).
- 후보 양식: `plugins/agentoppa/skills/self-harden/template.md`.
- **배포 안 됨** — `plugins/agentoppa/`만 실린다. `.agentoppa/`는 AgentOppa의 *자기* 개발 하네스(dogfood), repo가 자기 도구를 자신에게 굴리는 곳.
